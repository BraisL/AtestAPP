# ATESTAPP

Aplicación para la generación de atestados y otros documentos policiales.

## Versiones

### v0.2
- **Rediseño de la interfaz:** Se ha actualizado la cabecera para que sea más ancha y muestre el título de la aplicación de forma destacada.
- **Iconos con Font Awesome:** Se han añadido iconos de Font Awesome para mejorar la navegación y la experiencia de usuario.
- **Página de datos de la unidad:** Se ha creado una nueva página para gestionar los datos de la unidad, permitiendo al usuario introducir, guardar, cargar, borrar y descargar la información.
- **Persistencia de datos:** Los datos de la unidad se guardan en `localStorage` para que persistan entre sesiones.
- **Actualización dinámica:** La cabecera se actualiza para mostrar los datos de la unidad guardados.

### v0.1
- Estructura inicial del proyecto.
- Funcionalidad básica de la interfaz.
